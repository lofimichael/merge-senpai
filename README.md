This branch is intentionally a tiny media shelf. CI writes static review cards
here so Merge Senpai can demo durable artifacts without running a database,
bucket, or asset server.

Presentation note: this is a GitHub-flavored nod to the Infinite Storage Glitch
idea of using an unexpected host as storage:
https://github.com/KKarmugil/Infinite_Storage_Glitch
